Required:

- Console Emulator versão full (Caso no Windows)
- Adicionar o Console Emulator nas variáveis de ambiente
- Instalação Ruby+Devkit

Começando o projeto

 - bundle init (Caso não funcione, execute 'gem install bundler')
 - Adicionar gems no arquivo GemFile (Ex: "gem rspec")
 - bundle install (para baixar as dependencias cadastradas na linha acima)
 - rspec --init (para iniciar uma estrutura no projeto de testes como criar a pasta e arquivo spec)
 - Adicionar configurações do Capybara no spec_helper.rb 
 - Adicionar métodos do Capybara dentro do spec_helper.rb

Instalando o Chrome Driver

 - Pegar a versao do Chrome, o ChromeDriver precisa estar na mesma versao e realise
 - Acessar https://chromedriver.storage.googleapis.com/index.html?
 - Encontrar a mesma versao do navegador e baixar
 - No Windows, descompacte o arquivo e recorte o arquivo .exe para ser colado dentro da pasta C://DiscoC//Windows
